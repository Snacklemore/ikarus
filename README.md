# Running

## WebApp

1. Install https://www.meteor.com/
1. `cd ikarus-webapp/src`
1. `meteor`
1. Open http://localhost:3000/
1. Configure steam login by following instructions on the page.

### Tests

1. `cd ikarus-webapp`
1. npm install
1. npm test
